## 🧠 About The Project

This single-page portfolio showcases my journey, projects, certificates, and skills. It's built with a refined, classy aesthetic – white base, gold accents, smooth animations, and custom micro-interactions. The site includes:

- Animated greeting overlay with multilingual "Hello" messages
- Custom cursor (dot + ring) with hover effects
- Hero section with typed.js dynamic text
- Vinyl record player (tonearm click to spin)
- Polaroid-style flip card (click to reveal fun facts)
- Certificates gallery with click-to-view lightbox
- Project cards with detailed modals
- SVG‑animated timeline (scroll‑revealed)
- Contact form powered by EmailJS
- Fully responsive design

## 🛠️ Built With

- **HTML5** – semantic structure
- **CSS3** – custom properties, flex/grid, animations, responsive design
- **JavaScript (Vanilla)** – DOM manipulation, Intersection Observer, custom cursor, lightbox, etc.
- **Typed.js** – hero typing effect
- **EmailJS** – contact form without backend
- **Google Fonts** – Cormorant Garamond + DM Sans

## 🚀 Features

### 🧩 Interactive Elements
- **Greeting overlay** – cycles through 12 languages, then swipes up to reveal the site.
- **Custom cursor** – follows mouse, expands on hoverable elements.
- **Typed headline** – changes roles automatically.
- **Vinyl player** – click the tonearm to spin the record.
- **Polaroid card** – click to flip and read a quote + fun fact.

### 📜 Certificates Section
- Grid of certificate cards.
- Click any card to open a full‑image lightbox.
- Hover overlay with “View Full ↗”.

### 💼 Projects Section
- Three featured projects (Python scripts, Class Website, this portfolio).
- Each card expands into a modal with description, tech stack, and live link (where available).

### 🕰️ Experience Timeline
- SVG path connecting events.
- Each timeline item appears on scroll.
- Responsive: on mobile the SVG hides and items stack vertically.

### 📬 Contact Form
- Validates name, email, message.
- Sends email via EmailJS.
- Toast notification on success.
- Visual status messages.

## 📱 Responsive

- Desktop: full layout with hero right‑side vinyl visual.
- Tablet: certificates/projects switch to 2‑column grid, hero image hidden.
- Mobile: single‑column layout, collapsible navigation menu.
